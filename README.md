# Inky Renderer
A remote rendering service for Inkplate devices via Cloudflare Workers.

### This is a WIP!

### Image Services
1) Unsplash  (unsplash)
2) Wallhaven (wallhaven)

### Render Services
1) NY Times (news, nytimes)

### Supported Devices
1) Inkyplate 10
###### Plans for Inkyplate 6COLOR in the works.

2) Copy `config.example.json` to `./data/config.json`.
